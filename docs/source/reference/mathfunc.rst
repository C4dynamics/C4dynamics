.. _mathfunc:

**************
Math Functions
**************

.. currentmodule:: c4dynamics

For convenience, C4dynamics offers some of NumPy's trigonometric functions, 
with slight variations in some of them:


.. data:: sin 

    ``c4dynamics.sin = numpy.sin``

.. data:: cos

    ``c4dynamics.cos = numpy.cos``

.. data:: tan

    ``c4dynamics.tan = numpy.tan``

.. data:: asin

    ``c4dynamics.atan = numpy.arcsin``

.. data:: acos

    ``c4dynamics.atan = numpy.arccos``

.. data:: atan

    ``c4dynamics.atan = numpy.arctan``

.. data:: atan2

    ``c4dynamics.atan = numpy.arctan2``
