Datasets 
========
.. currentmodule:: c4dynamics.datasets 

The c4Dynamics datasets module provides a streamlined interface 
for fetching, caching, and managing datasets and pre-trained model files, 
supporting experimentation with c4Dynamics examples and use cases.
