c4dynamics.datapoint.inteqm#

datapoint.inteqm(forces, dt)#

Advances the state vector, datapoint.X, with respect to the input forces on a single step of time, dt.

Integrates equations of three degrees translational motion using the Runge-Kutta method.

This method numerically integrates the equations of motion for a dynamic system using the fourth-order Runge-Kutta method as given by int3.

The derivatives of the equations are of three dimensional translational motion as given by eqm3

Parameters#

forcesnumpy.array or list

An external forces vector acting on the body, forces = [Fx, Fy, Fz]

dtfloat

Interval time step for integration.

Returns#

outnumpy.float64

An acceleration array at the final time step.

Examples#

A Helium balloon of 100g weight floats with lift force of L = 0.05N and expreiences a side wind of 10k speed.

>>> t1, t2, dt = 0, 10, 0.01
>>> F = [0, 0, .05]
>>> hballoon = c4d.datapoint(mass = 0.1, vx = 10 * c4d.k2ms)
>>> for t in np.arange(t1, t2, dt):
...    hballoon.inteqm(F, dt)
...    hballoon.store(t)
>>> hballoon.draw('side')
>>> plt.gca().invert_yaxis()
