Constants#

Globals and conversion constants.

C4dynamics includes several constants as global quantities and conversion units:

Global Constants#

c4dynamics.utils.const.pi#

The ratio of a circle’s circumference to its diameter. C4dynamics’ pi is a simple assignment of numpy.pi:

pi = 3.1415926535897932384626433...

Reference

https://en.wikipedia.org/wiki/Pi

c4dynamics.utils.const.g_ms2#

Gravity of earth in meter per square second.

g = 9.80665

Reference

https://en.wikipedia.org/wiki/Gravity_of_Earth

c4dynamics.utils.const.g_fts2#

Gravity of earth in foot per square second.

g = 32.1740

Reference

https://en.wikipedia.org/wiki/Gravity_of_Earth

Conversion Constants#

c4dynamics.utils.const.ft2m#

foot to meter.

ft2m = 0.3048

c4dynamics.utils.const.lbft2tokgm2#

pound square foot to kilogram square meter.

lbft2tokgm2 = 4.88243

c4dynamics.utils.const.r2d#

radians to degrees.

r2d = 57.2958

c4dynamics.utils.const.d2r#

degrees to radians.

d2r = 0.0174533

c4dynamics.utils.const.k2ms#

knots to meters per second

k2ms = 0.514444
